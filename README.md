dom-edge
========

DOM技术研究, 高性能Web构建探究, 前沿技术探讨

全面的透彻前端技术，此项目 目的是

### 探究各种前沿技术，在企业级开发中的最佳实践。

### 还有针对部门的DOM培训，写的一些小例子

## Get
```bash  
$ git clone https://github.com/starzou/dom-edge  
$ cd dom-edge  
```

## Install
```bash  
$ npm install && bower install
```

## Usage
```bash  
$ npm start
```
